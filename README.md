# Java-Learn


## 【1】 文件拷贝程序

### 编译
在项目文件夹下面 
```
mvn assembly:assembly
```
生成的 `jar` 文件位于 `target` 文件夹下

### 运行
```
java -jar ./xxxx.jar
```
*更改了复制方式，使用文件管道复制*
